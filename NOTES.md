# Javascript
**HTML**

**var**
var a=10
var a ==> declaration 
a=10 ==> initialization
redeclaration and reinitialization is possible
hoisting is possible with var 
Var is a functional scope
var is not block scope

**let**
let is a block scope and functional scope but basically it is a **block scope**
redclaration is not possible but reinitialization is possible

**const**
redclaration and reinitialization is not possible
const a block scope and functional scope but basically it is a **block scope**

practically const is prefered

# Equality operator
- Loose equality (==) ==> it is comparing only value
